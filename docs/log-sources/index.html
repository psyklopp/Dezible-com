Log Sources

Logs provide a detailed, chronological record of events which is necessary to investigate security incidents.

Security analysts make use of data from different sources - network traffic logs, host-based data (think of system logs), threat intelligence to assess such security incidents.

Without data flowing in, there is nothing to look at!
